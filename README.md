# PreemptiveMultitasking
Preemptive multitasking demo in 8086 assembler.
Use DosBox (https://www.dosbox.com/ ) and Turbo Assembler 2.x to build&run.

BUILD:
tasm demo.asm
tlink /t demo.obj

demo.com
